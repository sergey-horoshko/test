# Opes

### В проекте используется [PrimeVue](https://primevue.org/), [Tailwind](https://tailwindcss.com/), [PrimeIcons](https://primevue.org/icons/)

### Создать файл .env.local и записать данные

```sh
BASE_URL=/
VITE_BASE_API=
#VITE_BASE_API=https://prod.ru
#VITE_BASE_API=https://dev.ru
VITE_APP_TOKEN_KEY=token
```

## Авторизация

```sh
Логин:
Пароль:
```

## Project Setup

```sh
yarn
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Compile and Minify for Production

```sh
yarn build
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn lint
```
